# examplePuppeteer
Exemplo do uso do puppeteer


- Baixe as dependencias rodando o comando:
```
yarn
```
 ou
```
npm install
```

- Para obter os dados da página, digite:
```
node index.js
```

- Para iniciar o servidor:
```
npx lite-server
```

* É preciso estar na pasta raiz do projeto para rodar os comandos
